Custom Elementor Addon
Version 1.0.0

Description:
A custom Elementor addon widget with neon border effects and responsive design.

Usage:
1. Install and activate the plugin.
2. Open Elementor editor.
3. Find "Custom Widget" in the widgets panel and drag it onto your page.

Features:
- Neon border effects on hover.
- Customizable color schemes and fonts.
- Responsive design.
